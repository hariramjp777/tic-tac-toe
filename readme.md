## Tic-Tac-Toe

* Fork the repository

* Clone the repository

```bash
git clone https://github.com/<username>/tic-tac-toe.git
```

* Go to the directory `tic-tac-toe`

```bash
cd tic-tac-toe/
```

* Compile the file `main.c` and create the executable `main`

```bash
gcc main.c -o main
```

* Run the game `main`

```bash
./main
```
